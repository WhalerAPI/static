This directory contains icons for development tools, build systems, and utilities.

- Yarn
- pnpm
- Webpack
- Vite
- Parcel
- Rollup
- Babel

- Prettier
- Docker
- Kubernetes

## File Naming Convention:
Use lowercase names with hyphens for multi-word tools (e.g., `npm.svg`, `webpack.png`, `github-actions.svg`)
